# Duda's Bolão ⚽

A pool application for the 2026 World Cup. Create or join pools, make your guesses, and compete with friends and family.

## Features

- Google login
- Create and join pools via invite code
- Group stage guesses — pick which teams advance
- Knockout stage guesses — predict exact scores
- Live scoreboard with points per pool

## Tech Stack

- [React](https://react.dev/) + JavaScript
- [Vite](https://vitejs.dev/)
- [Firebase](https://firebase.google.com/) (Authentication, Firestore)
- [React Router DOM](https://reactrouter.com/)
